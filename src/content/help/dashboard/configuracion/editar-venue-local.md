## Antes de empezar

Editar el venue cambia datos del local, no de tu usuario. Esta seccion puede incluir informacion basica, contacto, imagenes, documentos fiscales, integraciones y opciones generales. Antes de guardar, confirma que estas en la sucursal correcta y que el cambio corresponde a ese local.

Algunos campos afectan canales publicos o procesos operativos. Por ejemplo, imagenes pueden mostrarse al cliente, documentos pueden afectar KYC o pagos, e integraciones pueden cambiar sincronizaciones externas.

## Pasos

1. Abre **Configuracion > Editar Local**.
2. Entra a **Informacion basica** para revisar nombre, direccion, telefono y datos generales.
3. Usa **Contacto e imagenes** para actualizar identidad visual o medios visibles.
4. Usa **Documentacion** para cargar archivos fiscales, KYC o documentos requeridos.
5. Abre **Integraciones** para revisar conexiones como Google u otros servicios.
6. Guarda cada cambio y valida en la vista donde impacta: cliente, pagos, reservaciones o reportes.

## Problemas frecuentes

Si no puedes editar, tu usuario no tiene rol administrativo. Si una imagen no cambia, revisa formato, peso y cache del navegador. Si un documento queda pendiente, espera revision o revisa si falta algun archivo requerido. Si una integracion falla, confirma que la cuenta externa sea la correcta y que tenga permisos suficientes.

## Validaciones despues de editar

Para datos publicos, revisa la experiencia del cliente o canal donde se muestre el local. Para documentos, confirma estado de revision y que no haya rechazos. Para integraciones, realiza una prueba concreta, como sincronizar una reserva, abrir un enlace o validar que la cuenta conectada sea la esperada.

Si el cambio afecta pagos, no lo pruebes con un cliente real primero. Usa un caso controlado o pide apoyo a soporte para validar configuracion.

## Cuando pedir ayuda

Escala si un documento aprobado aparece rechazado, si el local muestra datos de otra sucursal, si una integracion queda conectada a una cuenta equivocada o si no puedes guardar informacion critica. Incluye sucursal, campo afectado, fecha del cambio y captura antes/despues.
