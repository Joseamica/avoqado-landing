## Antes de empezar

Turnos ayuda a revisar aperturas, cierres y actividad del equipo. Se usa para seguimiento operativo y puede estar relacionado con caja, comisiones o reportes internos segun configuracion del local. Antes de corregir un turno, confirma si el error viene de horario, usuario, cierre o ventas asociadas.

No todos los roles pueden ver o modificar turnos. Si un encargado necesita operar esta seccion, revisa sus permisos de equipo y turnos.

## Pasos

1. Abre **Equipo > Turnos**.
2. Ajusta el periodo que quieres revisar.
3. Busca por miembro del equipo o estado del turno.
4. Abre el detalle para revisar apertura, cierre y actividad relacionada.
5. Compara contra ventas o pagos si estas investigando diferencias de caja.
6. Documenta cualquier correccion operativa para que el equipo tenga contexto.

## Problemas frecuentes

Si faltan turnos, revisa fecha, sucursal y filtros. Si un turno quedo abierto, valida quien lo opero antes de cerrarlo o corregirlo. Si los montos no coinciden, compara pedidos, pagos y comisiones. Si la seccion no aparece, puede estar deshabilitada para el local.

Cuando hagas una correccion, deja evidencia del motivo y del responsable para evitar diferencias al cierre administrativo.

## Uso para cierre operativo

Al cierre, revisa turnos abiertos, horarios de entrada y salida, ventas asociadas y cualquier diferencia reportada por el equipo. Si el local usa turnos para caja o comisiones, no ajustes datos sin confirmar primero con el responsable del turno.

Para auditoria, guarda fecha, usuario, hora de apertura, hora de cierre y motivo de cualquier correccion. Esto facilita explicar diferencias cuando se revisan pagos, pedidos o comisiones dias despues.

## Cuando pedir ayuda

Pide soporte si un turno no permite cerrarse, si un usuario no aparece en filtros o si los datos de turno no coinciden con actividad visible en ventas. Incluye usuario, fecha, sucursal y estado del turno.
